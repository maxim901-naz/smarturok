# SmartUrok / MaxSchool

Django-based online school platform.

## Local run
1. Create `.env` from `.env.example`.
2. Install dependencies: `pip install -r requirements.txt`
3. Run migrations: `python maxschool/manage.py migrate`
4. Start server: `python maxschool/manage.py runserver`

## Deployment
- Timeweb deployment guide: `DEPLOY_TIMEWEB.md`
- Production env template: `.env.prod.example`
